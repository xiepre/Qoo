# Railway Booth Quotation App

## Deploy to Railway without local CLI
1. Create a new GitHub repository.
2. Upload all files in this project to GitHub.
3. Go to Railway.
4. Choose **New Project** -> **Deploy from GitHub repo**.
5. Select this repository.
6. Railway will run:
   - `npm install`
   - `npm run build`
   - `npm start`

## Scripts
- `npm run dev` - local development
- `npm run build` - build frontend
- `npm start` - run Express server for Railway
